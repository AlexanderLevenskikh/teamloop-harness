# YourAITeam 0.4.0 — YourAITeam MVP

## Decision

`DYNAMIC_TEAM_COMPOSITION_MVP_PASS`

YourAITeam now has a task-specific staffing front door. It proposes a minimum sufficient team, estimates role and coordination tokens, negotiates explicit trade-offs, requires acceptance, and materializes the accepted team for Codex or OpenCode.

## Added

- deterministic task classifier and role composer;
- role grades: economy, balanced, premium;
- token range and coordination-overhead estimate;
- natural-language bargaining for common constraints;
- immutable accepted-team contract;
- Codex agent/skill materializer;
- OpenCode primary/subagent materializer;
- adapter-contract capabilities;
- six-scenario task matrix;
- worked end-to-end proposal/negotiation/materialization examples;
- 15 dedicated tests;
- market note and deep-research prompt;
- book chapter: `Когда метрикам понадобился менеджер`.

## Important boundary

This is a staffing and configuration MVP, not a billing system. Estimates are planning units and are not live backend token measurements. Codex and OpenCode CLIs were not present in the build container, so generated configurations were structurally validated but not executed by those products in this release run.
