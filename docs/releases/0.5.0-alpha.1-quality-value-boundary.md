# YourAITeam 0.5.0-alpha.1 — quality/value boundary

## Added

- deterministic boundary packets from primary artifacts;
- hard-invariant and soft-debt policy separation;
- six runtime-validated boundary decisions;
- read-only `quality-value-manager` role and three reusable skills;
- finite profile budgets and no-progress stopping;
- hash-chained decision/improvement history;
- fingerprint-bound acceptance receipts and advancement lock;
- restart-safe boundary state and dashboard status;
- generic software-task adapter;
- unit, integration, and adversarial tests.

## Safety boundary

The manager cannot override gates, edit evidence, alter budgets, or advance work by its decision alone. Existing scope, reviewer, sentinel, and final-gate protections remain active.

## Alpha limitations

- boundary contracts are opt-in rather than automatically generated for every task;
- the included dashboard is a dependency-free generated HTML/JSON status surface, not a persistent multi-workspace web service;
- domain adapters beyond generic software work remain future integrations;
- the internal `.teamloop` technical namespace remains for compatibility.
