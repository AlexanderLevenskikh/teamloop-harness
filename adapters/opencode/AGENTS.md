# TeamLoop Harness - OpenCode Adapter

Place this adapter's files into your OpenCode project:

1. Copy `opencode.jsonc` to project root as `opencode.jsonc`
2. Copy `AGENTS.md` to project root
3. Copy `adapters/opencode/agents/` to `.opencode/agents/`
4. Copy `adapters/opencode/commands/` to `.opencode/commands/`

Then use `/supervised-task` to start a supervised delivery run.
