# Quality/value arbitration

Use only an authoritative boundary packet.

- Hard checks define what is forbidden.
- Among allowed actions, maximize reusable expected payoff.
- Accept only with zero hard failures.
- Record every current soft-debt item when accepting with debt.
- Select exactly one authoritative candidate for improvement.
- Do not treat ticket closure, generated files, or stage visitation as user value.
